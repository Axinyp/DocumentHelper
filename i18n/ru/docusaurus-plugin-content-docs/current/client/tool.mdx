#  Как пользоваться гаджетами

​        Чтобы создать полноценный процесс проведения совещания, на совещании по внедрению используются гаджеты, а гаджеты можно использовать только на обычных совещаниях.

## 1. Быстрая передача файлов

 		Нажмите на боковые инструменты в функции быстрой передачи . После того, как она будет раскрытата, мобильным телефоном отсканируйте QR-код , чтобы загрузить файл, который вы хотите загрузить.Mожно загружать и удалять синхронно, а загруженный файл можно сохранить на локальном диске,  на собрании и на USB-накопителе.Нажмите на кнопку уменьшения в правом верхнем углу QR-кода, чтобы уменьшить файл и быстро загрузить его на страницу собрания, и вы все еще cможете продолжить загрузку файла.

## 2. Голосование

​		Откройте окно голосования, нажав на боковое поле, чтобы проголосовать.Голосование делится на быстрое голосование и расширенное пользовательское голосование.

➤Типы голоса для быстрого голосования делятся на голос по голосованию и голос за удовлетворение. После выбора типа голосования вы можете отсканировать код для голосования или поднять руку для голосования в настройках голосования.После завершения выбора нажмите на кнопку Создать голосование, чтобы проголосовать.

<img  src={useBaseUrl("img/doc-zh/投票-zh.png")} />

➤Расширенное пользовательское голосование

​		Расширенное голосование - это форма голосования, выбранная путем сканирования кода для настройки. После выбора расширенного голосования отсканируйте QR-код с помощью своего мобильного телефона, чтобы настроить его. Когда настройка будет завершена, QR-код для голосования отобразится на моноблоке, и  можно участвовать в пользовательском голосовании путем сканирования QR-кода.

<img  src={useBaseUrl("img/doc-zh/高级投票-zh.png")} />


➤Итоги голосования

​		После формирования иготов голосования их можно просмотреть в графическом формате, а если выбран неанонимное голосование, то можно просмотреть и подробную информацию. Итоги голосования можно вставить на доску, сохранить на совещании, локально, на USB-накопителе, отсканировать и забрать с собой.

## 3. Оценка

​         Откройте оценку через боковое поле с обеих сторон. Метод оценки - пользовательская оценка. Оценка производится после установки опоции оценки путем сканирования кода.

## 4. Регистрация

​       Нажмите на кнопку"Регистрация" на боковом поле и QR-код отобразится после открытия.После сканирования кода для регистрации QR-код находится слева, а количество регистрации и имена - в правой части интерфейса.Вы можете [повторно зарегистрироваться] или [отсканировать код, чтобы получить список].

## 5. Запись экрана

​        Запись экрана можно загрузить из магазина приложений инструментов на боковом поле. Запись экрана делится на 2K и 4K, 2K - это разрешение 1080P для обычного монитора, 4K - разрешение 3840. Можно выбрать, открывать ли микрофон, при начале записи экран через 5 секунд автоматически сжимается в нижнем правом углу в красную точку с логотипом для отсчета времени; после окончания записи можно выбрать место сохранения: на собрании, локальном диске или на USB-накопителе.В режиме 2K запись может длиться четыре часа, в режиме 4K - два часа.

​		 После завершения записи нажмите на кнопку “ Завершить запись” сначала будет загружен QR-код, полученный путем сканирования кода.Если вы хотите сохранить его на собрании, вам нужно дождаться полной загрузки QR-кода.Если вы просто хотите сохранить его на локальном диске, просто нажмите на кнопку “Сохранить”, не нужно ждать загрузки QR-кода.

## 6. Скриншот

​        Скриншот можно загрузить из магазина приложений инструментов на боковом поле. После загрузки он будет открыт,можно увеличивать, уменьшать масштаб,перетаскивать
четыре угла поля выбора, чтобы выбрать любую область. Содержимое скриншота можно также выбрать для места сохранения , можно напрямую поделиться, можно также напрямую вставить в доску.

## 7. Стена для подписей

​       	Стена для подписей можно загрузить из магазина приложений инструментов на боковом поле. Стена подписей - это панель для письма, предназначенная для объединения некоторых функций фотосъемки и белых досок, позволяющая делать несколько подписей и фотографий.Нажмите на кнопку "Настройки", чтобы задать цвет и размер шрифта приветственного слова на стене для подписей. После добавления фонового изображения выберите и нажмите на кнопку"Сохранить", чтобы изменить фон стены для подписей.Вы можете настроить включение и выключение камеры, а также ее переворот.Изображение можно увеличивать и уменьшать после съемки, а также сделать скриншот, чтобы забрать его с собой или сохранить в коллекции подписей.

## 8. Таймер

​      	После загрузки через магазин приложений и открытия таймера вы можете установить время для отсчета времени. После установки времени, если в течение 5 секунд не будет произведено никаких действий, таймер автоматически уменьшится в правом верхнем углу, а по истечении установленного времени он будет отображаться красным цветом.Интерфейс таймера может быть расширен для установки времени.
