## 1 Условия проецирования на экран NFC

Для поддержки прогнозирования на экран NUF необходимо выполнить все следующие три условия：

1.Мобильный телефон должен иметь функцию NFC
2.На мобильном телефоне должна быть включена функция NFC.
3.На мобильном телефоне должно быть установлено программное обеспечение для прогнозирования на экран.

## 2  Способ использования

1.После включения функции NFC на мобильном телефоне (если на мобильном телефоне не используется NFC, необходимо открыть его вручную).
2.Используйте заднюю крышку телефона, чтобы держать ее поближе к наклейке NFC.
3.Телефон будет индуктивно вибрировать.Если на мобильном телефоне не установлено программное обеспечение для  проецирования на экран, мобильный телефон автоматически перейдет к интерфейсу загрузки.
4.После загрузки телефон может транслировать экран, прикоснувшись к другой наклейке NFC.
