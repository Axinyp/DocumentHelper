## 1. Как начать создавать примечание

Существует два способа открыть примечания: один - нажать на кнопку  "Примечание" непосредственно на боковом поле, а другой - сделать примечание пером ниже при просмотре документа.Это позволяет пользователям написать контент и комментировать информацию на текущем экране.

## 2. Область примечания

Область применения примечания - глобальное примечание, примечание и белая доска являются взаимоисключающими, и открыть можно только одну из них.
