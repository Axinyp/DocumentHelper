1.Когда администратор входит в управление умной конференц-платформой, открывает раздел [Управление комнатами] и выбирает [Управление оборудованием]. Затем нажимает на кнопку "+" в области комнат для добавления новой конференц-комнаты.

<img  src={useBaseUrl("img/doc-zh/激活新增会议室-zh.png")} />

2.После добавления новой конференц-комнаты, выберите [Создать код активации] и выберите соответствующую комнату. Подтвердите выбор, чтобы сгенерировать код активации.
