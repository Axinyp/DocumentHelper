## 1.1 快速会议

​         快速会议随想随开，点击快速会议可以立即发起一个会议。当点击结束会议后，不能在首页会议列表找到这个会议，但可以在后台查询此记录。若发生意外如突然断电，开机后可在首页的会议列表找到未结束的会议，点击即可继续会议或者结束会议。快速会议中可使用投票、评分、签到、文件快传等一些工具。

## 1.2 保密会议

​        保密会议即操作痕迹保密，会中产生的资料不可以分享和保存。保密会议中不可以使用截屏和录屏功能，小工具均具有保密功能，不可以分享保存扫码带走。会中全程附带水印效果，摄像头可拆卸，实现画面和声音的保密效果。
<img  src={useBaseUrl("img/doc-zh/保密会议-zh.png")} />

## 1.3 预约会议

​         可提前预约会议。可以预约一个时间段的会议，预约列表只显示预约在当天的会议。点击会议即可进入开会。如果没有待开始和未结束会议时，首页不显示会议列表。预约会议列表可以显示待开始的预约会议、已结束的预约会议、因意外例如断电而中断的会议。
