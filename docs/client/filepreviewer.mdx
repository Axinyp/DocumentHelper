## 5.1  File manager

​        Click the floating menu bar on both sides to open the file manager. The file manager will display the local files and the files inserted into the USB flash drive. When you click edit in the lower right corner or press and hold a file to enter the editing state, a selectable state will appear, and you can operate the file, rename a single file, move, copy, and delete a single or multiple files. Click All/Documents/Pictures/Audio/Video in the upper right corner to divide the files. When the selected category finds no files, you can check whether the file path in the upper left corner is correct. The buttons in the upper right corner are the top button and the close button.

## 5.2  File display

​        After selecting a file to open in the task manager, a task bar will appear below. Depending on the file type, the bottom toolbar can open the whiteboard, rotate, switch pictures to the previous or next picture, preview all pictures, insert materials, etc.

<img  src={useBaseUrl("img/doc-en/文件展示-en.png")} />

## 5.3 Multiple file display

​        File display can open multiple files at the same time. The format is audio, picture, video, document, etc. Up to four files can be open at the same time.

<img  src={useBaseUrl("img/doc-en/多文件-en.png")} />

## 5.4  PPT speech mode

​        The presentation mode has been added to the PPT file. The speech mode is divided into regular demo mode and timed speech mode. The normal speech mode is to open the PPT normally. The timed speech mode is to perform a timed speech by customizing the time. After setting the time, click Enable, and the time will be counted in the form of a small window in the upper right corner. Click on the timing widget to pause or resume the time.
